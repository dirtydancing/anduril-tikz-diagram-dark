available over there.

The corresponding two commands are:

`lualatex anduril-tikz-diagram-dark.tex`

`pdftocairo -png -singlefile -r 300 -W 3507 -H 2480 anduril-tikz-diagram-dark.pdf anduril-tikz-diagram-dark`
